Test of LigerBots URL Shortening Service

This is a simple URL shortening service that redirects a browser when it receives a request for a URL with a "short code" to another longer url.

Short code URLs take two forms:

  • /u/XXXXX where XXXXX is a short code that is used to redirect to the long URL, and
  • /u/XXXXX/m which is the same but with a trailing /m to indicate that the short url was clicked from an email message.

For example, the long URL http://www.ligerbots.com/ has the short code 1q1q1q which is used in the URL http://sandbox.0mo.cc/u/1q1q1q.

You can try it here: LigerBots Home

This service also "drops a cookie" into the browser of the user to uniquely identify her. Once the cookie is set, the service will track the number of times the user clicks on short URLs.

This service provides a simple API which is used to view the short->long URL mappings, and to create new short URLs. The API is:

  • GET /api/redirects to get a list of all short codes and their matching long URLs
  • GET /api/redirects/:id to get the long URL for a specific short code
  • POST /api/redirects to create a new short URL. The body of this request can either be:
    • { "url": "target long url" } in which case the service will create a new random short url, or
    • { "id": "short code", "url": "target long url" } in which case the service will create a mapping between the short code and the target URL.
