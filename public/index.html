Test of LigerBots URL Shortening Service

This is a simple URL shortening service that redirects a browser to a long url when it receives a request for a short url.

Short URLs take two forms:

  • /u/XXXXX where XXXXX is a short code that is used to redirect to the long URL, and
  • /u/XXXXX/m which is the same with a trailing /m that indicates the short url originated in an email message.

For example, the long URL http://www.ligerbots.com/ has the short URL http://sandbox.0mo.cc/u/1q1q1q.
